# Diabetes-Prediction

We do :

  - Preprocess Data
  - Learning & Evaluation

We try many methods like `K-Nearest-Neighbours`, `Decision‬‬ ‫‪Trees`, `Logistic‬‬ ‫‪Regression‬‬` and `Random Forest`.
